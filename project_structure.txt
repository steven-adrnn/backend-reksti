Proposed FastAPI Backend Project Directory Structure:

backend/
├── app/
│   ├── __init__.py
│   ├── main.py                  # FastAPI app entrypoint
│   ├── config.py                # Configuration settings (Supabase, secrets, etc.)
│   ├── models.py                # Pydantic models and database models
│   ├── database.py              # Supabase/PostgreSQL client setup
│   ├── auth.py                  # User authentication and authorization logic
│   ├── websocket.py             # WebSocket server handling (ESP32-CAM and mobile clients)
│   ├── detection.py             # ML human detection logic (YOLO integration)
│   ├── recording.py             # Video recording and saving logic
│   ├── supabase_client.py       # Supabase storage and metadata integration
│   ├── api/
│   │   ├── __init__.py
│   │   ├── endpoints.py         # REST API endpoints for mobile app (login, video list, signed URL)
│   └── utils.py                 # Utility functions
├── requirements.txt             # Python dependencies
├── Dockerfile                   # For containerized deployment
├── README.md                    # Project overview and setup instructions
└── tests/
    ├── test_auth.py
    ├── test_websocket.py
    ├── test_detection.py
    └── test_api.py
